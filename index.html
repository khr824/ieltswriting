Task 1 Tips
  • Paraphrase the Introduction: Never copy the prompt word-for-word. Use synonyms.
  • The 'Overview' is King: You MUST include a general overview paragraph describing the main trends. No data in this part!
  • No Opinions: Never write "I think" or give reasons for the data unless shown in the chart.
  • Group Data: Don't list every number. Group similar trends together (e.g., "Both UK and USA saw an increase...").
Task 2 Tips
  • Plan before you write: Spend 5 minutes brainstorming. A clear structure saves time later.
  • One Idea Per Paragraph: Start with a clear Topic Sentence, explain it, then give an example.
  • Clear Position: If asked for your opinion, state it clearly in the introduction and conclusion.
  • Time Management: Leave 3-5 minutes at the end to check for spelling and basic grammar errors.
